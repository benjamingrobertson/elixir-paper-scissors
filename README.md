## Installation

Run `mix escript.build`

From the command line, run `./roshambo` and pass in your pick (rock, paper, scissors).

`./roshambo paper`

You can pass in a second parameter, preceded by --, to tell the computer what its move should be.

`./roshambo paper --rock`.
